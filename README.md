# nvim config

my personal neovim setup. mostly focused on java development and keeping things fast/minimal.

## dependencies
- neovim >= 0.10.0
- ripgrep (for telescope)
- a nerd font
- git, make, gcc (for treesitter)

## install
```bash
mv ~/.config/nvim ~/.config/nvim.bak
git clone <your-repo-url> ~/.config/nvim
```

open nvim and let lazy install everything.
